📋 HOW TO USE
================================================================================

1. START FRONTEND:
   cd frontend
   npm install
   npm run dev
   # Access: http://localhost:5173

2. LOGIN:
   Username: admin or dispatcher
   Password: any password (demo mode)

3. NAVIGATE:
   - Dashboard: Operations Center
   - Tracking: Live GPS Fleet Map
   - Predictions: AI Analytics
   - Fleet: Vehicle & Driver Directory
   - Emergencies: Incident Management

4. BUILD FOR PRODUCTION:
   npm run build
   npm run preview
